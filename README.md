# Claude Code Workshop

Build apps by chatting with Claude. No install needed — runs in your browser.

## Launch

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/diganelin/claude-code-intro?quickstart=1)

Click the button. Sign into GitHub if it asks. Wait ~60 seconds while your workspace starts up.

## Start building

Once the editor loads in your browser:

1. Open the terminal (bottom panel — or press `` Ctrl+` ``)
2. Type `claude` and press Enter
3. If it asks "Do you want to use this API key?" → pick **Yes** (option 1). It defaults to "No" but Yes is correct here.
4. Tell Claude what you want to build

That's it. Claude writes the code, you run it, you change your mind, you iterate.

## What can I build?

Anything that runs in a terminal or shows up in a browser tab. Some starting points:

- **Streamlit apps** — quick dashboards, mini-tools, interactive charts
- **Websites** — plain HTML/CSS/JS, or Flask / FastAPI in Python
- **Command-line toys** — games, scripts, utilities
- **Notebooks** — Jupyter for data stuff
- **Anything using an API** — weather, music, sports scores, memes

Peek at the `exercises/` folder for inspiration.

## Keeping your work

Files auto-save, but your Codespace is a cloud computer that goes away eventually. If you build something you want to keep, ask Claude to help you push it to your own GitHub repo.

## Running into trouble?

- Terminal stuck? Click the `+` in the terminal panel to open a new one, or press `Ctrl+C` to stop what's running.
- App not showing up? Look at the **Ports** tab in the bottom panel — click the globe icon next to your port.
- Claude acting weird? Type `/clear` to reset the conversation, or just close the terminal and type `claude` again.
