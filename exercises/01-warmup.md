# 1. Warm up

Get a feel for talking to Claude. Try a few of these:

- "What files are in this folder?"
- "Write a Python script that prints a poem about the sea, then run it."
- "Make a file called `notes.md` with 5 things I should know about Python."
- "What's the weirdest one-line Python program you can write?"

Goal: get comfortable with the back-and-forth. Claude writes, runs, shows results. You react.
